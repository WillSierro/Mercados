# Sakashita MVP

Protótipo mobile-first para a Sakashita divulgar ofertas do dia em um canal proprio.

## O que ja funciona

- Tela de ofertas para o cliente
- Busca por produto
- Filtro por categoria e destaque
- Troca entre entrega e retirada
- Escolha de loja
- Painel para cadastrar, editar, pausar e excluir ofertas
- Persistencia local no navegador com `localStorage`

## Como usar

1. Abra `index.html` no navegador.
2. Use `Painel` para cadastrar ou editar ofertas.
3. Volte em `Ofertas` para ver a vitrine do cliente.

## Arquivos

- `index.html`: estrutura da interface
- `styles.css`: identidade visual e responsividade
- `script.js`: regras da aplicacao e persistencia local
